## ig start
The `ig start` command builds the application, starts a web server and opens the application in the default browser.

## Arguments
<details>
  <summary>port <span align="right"><strong>(v2.0.0)</strong></span></summary>
  <p>
    <code>--port</code> (alias: <code>-p</code>) <em>default value: defaultPort specified in the project specific ignite-ui-cli.json file.</em>
  </p>
  <p>
     Can optionally pass a port to be used when serving the application.
     <br> 
     <code>
        ig start --port=1234
     </code>
     <br>
     or using aliases:
     <br> 
     <code>
     ig start -p=1234
     </code>
  </p>
</details>
